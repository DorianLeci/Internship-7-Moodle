# Internship-7-Moodle

Installation

To run this console app you need to have the following installed on your computer:
.NET SDK. for .NET CLI(command line interface). It is availible on https://dotnet.microsoft.com/en-us/download.
This project was made using .NET 8.0.

Steps to install the project:
```
git clone git@github.com:DorianLeci/Internship-7-Moodle.git
cd Internship-7-Moodle
cd Internship-7-Moodle
cd Internship-7-Moodle.Presentation
```

After positioning in the right folder type

```
dotnet run
```
